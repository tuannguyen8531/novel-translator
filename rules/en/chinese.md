# Chinese -> English — Language-Specific Rules

## Names
- Use readable romanization for personal names unless the glossary specifies a different established rendering
- Keep given name and family name order consistent with the chosen novel style
- Place names, sect names, organizations, realms, and technique names may be translated by meaning or romanized; prioritize consistency and readability
- Do not convert names into Vietnamese-style Hán Việt when the target is English

## Pronouns and Address
- 我 -> I / me, but preserve arrogance, humility, or intimacy through diction when relevant
- 你 -> you, but reflect respect or contempt through tone, title, or word choice
- 他/她/它 -> he / she / it / they according to context; avoid guessing gender when the source is ambiguous and context does not resolve it
- 老子, 本座, 本王, 朕, 在下, 小的 and similar self-references should become natural English voice markers, not literal glosses

## Honorifics and Titles
- 前辈 -> Senior / elder / predecessor, depending on context
- 师父 -> Master / Teacher
- 师兄 / 师姐 / 师弟 / 师妹 -> Senior Brother / Senior Sister / Junior Brother / Junior Sister when sect hierarchy matters
- 大人 -> Lord / Your Excellency / Sir / Madam, depending on setting and status
- 兄弟 -> brother / buddy / sworn brother, depending on relationship
- 老师 -> Teacher / Professor / Mr. / Ms., depending on setting
- 陛下 / 殿下 -> Your Majesty / Your Highness
- 公子 / 少爷 / 小姐 -> Young Master / Young Lord / Miss / Lady, depending on setting

## Xianxia / Xuanhuan Terms
- 修炼 -> cultivate / train / practice cultivation
- 境界 -> realm / stage
- 灵气 -> spiritual energy / qi, depending on glossary
- 丹药 -> pill / elixir
- 法宝 -> magical treasure / artifact
- 功法 -> cultivation method / technique
- 突破 -> break through / breakthrough
- 渡劫 -> undergo tribulation / pass the heavenly tribulation
- 内力 -> internal force
- 真气 -> true qi
- 元神 -> primordial spirit / nascent soul, depending on genre context
- 神识 -> divine sense / spiritual sense

## Wuxia Terms
- 江湖 -> jianghu / martial world, depending on glossary and readability
- 武功 -> martial arts / martial skill
- 剑法 -> sword technique / sword art
- 掌门 -> sect leader / clan master
- 门派 -> sect / school / faction
- 轻功 -> qinggong / lightness skill
- 内功 -> internal art / internal cultivation
- 大侠 -> hero / great hero / martial hero

## Modern / Campus Novel Terms
- 校长 -> principal
- 导员 / 辅导员 -> counselor / academic advisor
- 学长 / 学姐 -> senior / upperclassman / senior schoolmate
- 贴吧 -> forum / Tieba, depending on context
- 传单 -> flyer / leaflet
- 班主任 -> homeroom teacher / class adviser
- 同桌 -> deskmate / seatmate
- 同学 -> classmate
- 学霸 -> top student / academic ace
- 学渣 -> poor student / academic underachiever
- 社团 -> club / society
- 校服 -> school uniform
- 月考 / 期中考 / 期末考 -> monthly exam / midterm / final exam

## Youth / Campus Novel Style
- Keep school-life dialogue youthful, natural, and lightly witty when the source tone allows it
- Preserve awkwardness, crush tension, teasing, and small daily details
- Do not make teenagers sound like formal adults unless the source deliberately does
- Avoid over-romanizing ordinary school terms when a natural English term exists

## Idioms and Classical Flavor
- Translate idioms by meaning; preserve the literal image only when it is important to the scene
- For classical, martial, imperial, or sect settings, use elevated but readable English
- Do not overload the translation with archaic English unless the source is deliberately archaic
- Keep parallel phrasing, proverbs, and poetic imagery when they carry style or emotional weight
