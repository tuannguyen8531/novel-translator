# Japanese -> English — Language-Specific Rules

## Names
- Keep Japanese names romanized consistently unless the glossary specifies otherwise
- Choose one ordering style for the novel and keep it consistent: surname-first for Japanese flavor, or given-name-first if the glossary establishes it
- Do not convert Japanese names to Chinese-style or Vietnamese-style readings
- Keep organization, school, class, guild, and noble house names consistent with the glossary

## Name Suffixes and Address
- ~さん (san): Mr. / Ms. / first-name address / retained -san, depending on setting and glossary
- ~様 (sama): Lord / Lady / Master / Your Grace / retained -sama when the tone depends on it
- ~くん (kun): first-name address, junior address, or retained -kun when school/social nuance matters
- ~ちゃん (chan): affectionate diminutive, nickname, or retained -chan when character voice depends on it
- ~先生 (sensei): teacher / professor / doctor / master, depending on context
- ~先輩 (senpai): senior / upperclassman / retained senpai when setting depends on it
- ~殿 (dono): Lord / Sir / formal address, depending on period and tone
- Do not retain every suffix mechanically; preserve the social effect in natural English

## Keigo (Politeness Levels)
- 丁寧語 (teineigo): use polite, measured English
- 尊敬語 (sonkeigo): use respectful phrasing, titles, and deferential tone
- 謙譲語 (kenjougo): use humble phrasing without becoming awkward or archaic
- Casual speech should sound direct and natural, not like formal translated Japanese
- Preserve abrupt switches in politeness when they signal anger, intimacy, panic, or social rupture

## Isekai / Light Novel Terms
- 異世界 -> another world / otherworld / isekai, depending on glossary
- 勇者 -> hero / chosen hero
- 魔王 -> Demon King / Demon Lord
- スキル -> skill
- レベル -> level
- ステータス -> status
- ダンジョン -> dungeon
- 冒険者 -> adventurer
- ギルド -> guild
- 魔法 -> magic
- 魔物 -> monster / magical beast
- 聖女 -> saint / holy maiden
- 騎士 -> knight
- 召喚 -> summon / summoning
- 転生 -> reincarnation
- 転移 -> transfer / teleportation / transported to another world, depending on context

## Academy / Modern / Slice-of-Life Terms
- 学園 / 学校 -> academy / school
- 先生 -> teacher / professor / doctor, depending on context
- 先輩 / 後輩 -> senior / junior / senpai / kouhai, depending on glossary
- 委員長 -> class representative / class president
- 生徒会 -> student council
- 部活 -> club activity / club
- 幼馴染 -> childhood friend
- 告白 -> confession / love confession, depending on context

## Onomatopoeia and Sound Effects
- Translate meaning, not sound, when the sound effect carries description
- ドキドキ -> heart pounding / thumping
- バタバタ -> flurry / clatter / running around, depending on scene
- シーン -> silence / dead silence
- じっと -> intently / still / fixedly, depending on context
- Preserve short sound effects only when they read naturally in English or are part of a comic/light-novel style

## Style Notes
- Light novels often use first-person internal monologue; preserve the introspective rhythm
- Keep humor, self-deprecation, and overreaction lively without overexplaining
- Avoid stiff literal renderings of particles, sentence endings, and stock phrases
- Preserve contrast between narration, inner monologue, and spoken dialogue
- Use natural English sentence flow even when the Japanese source uses long modifier chains

## Cultural and Register Notes
- Retain Japanese honorifics only when the story's tone, setting, or glossary depends on them
- Do not insert translator notes for food, school systems, festivals, guilds, or honorifics
- Translate jokes by function when a literal rendering would not work in English
- Preserve otaku/game/system terminology consistently with the glossary
