# Chinese → Vietnamese — Language-Specific Rules

## Names
- Translate character names to Hán Việt when appropriate (e.g., 李明 → Lý Minh, 陆远秋 → Lục Viễn Thu)
- Place names: use Hán Việt (e.g., 青云山 → Thanh Vân Sơn)

## Pronouns (context-dependent)
- 我 → tôi / ta / mình (depends on character's personality and scene)
- 你 → cậu / ngươi / anh / em (depends on relationship)
- 他/她 → hắn / y / anh ấy / cô ấy (depends on tone)

## Honorifics
- 前辈 → tiền bối
- 师父 → sư phụ
- 大人 → đại nhân
- 兄弟 → huynh đệ
- 老师 → thầy / cô (depends on gender)

## Xianxia / Xuanhuan Terms
- 修炼 → tu luyện
- 境界 → cảnh giới
- 灵气 → linh khí
- 丹药 → đan dược
- 法宝 → pháp bảo
- 功法 → công pháp
- 突破 → đột phá
- 渡劫 → độ kiếp
- 内力 → nội lực
- 真气 → chân khí

## Wuxia Terms
- 江湖 → giang hồ
- 武功 → võ công
- 剑法 → kiếm pháp
- 掌门 → chưởng môn
- 门派 → môn phái

## Modern / Campus Novel Terms
- 校长 → hiệu trưởng
- 导员/辅导员 → cố vấn học tập
- 学长/学姐 → học trưởng / học tỷ
- 贴吧 → diễn đàn
- 传单 → tờ rơi

## Youth / Campus Novel Style
- 校园 / 青春校园 / 青春文学 → thanh xuân vườn trường / truyện học đường
- 班主任 → chủ nhiệm lớp
- 同桌 → bạn cùng bàn
- 同学 → bạn học / bạn cùng lớp
- 学霸 → học bá / mọt sách giỏi
- 学渣 → học tra / học dở
- 社团 → câu lạc bộ / CLB
- 校服 → đồng phục học sinh
- 早读 → giờ đọc buổi sáng
- 月考 / 期中考 / 期末考 → kiểm tra tháng / thi giữa kỳ / thi cuối kỳ
- Do not make school-life dialogue sound too formal; keep it youthful, natural, and lightly witty when the source tone allows
- For everyday campus scenes, prefer plain Vietnamese over Hán-Việt-heavy phrasing unless the source is clearly literary or nostalgic
- Preserve the emotional awkwardness, crush tension, teasing, and small daily details typical of youth novels
